    • ۱۲ میانبر برای هدایت و انتخاب متن در مک !!




    یکی از مشکلاتی که کاربران تازه‌کار مک با اون مواجه هستن نبود دکمه‌های Home و End و Page Up و Page Down کیبورد ویندوز در
    کیبورد مک است. بخصوص اگر تایپیست باشید یا کار شما با تایپ زیاد سر و کار داشته باشد عادت کردن به کیبورد مک برای شما
    خیلی سخت خواهد بود.





    اما راه حل‌هایی هم برای رفع این مشکل در مک وجود داره مثل این ۱۲ میانبری که با هم مرور می‌کنیم:

    1. Command + Left Arrow برای رفتن به ابتدای خط
    2. Command + Right Arrow برای رفتن به انتهای خط
    3. Option + Right Arrow برای رفتن به ابتدای کلمه‌ای که در حال تایپ اون هستید
    4. Option + Left Arrow برای رفتن به انتهای کلمه‌ای که در حال تایپ اون هستید
    5. Command + Up Arrow برای رفتن به ابتدای متن
    6. Command + Down Arrow برای رفتن به انتهای متن
    حال با اضافه کردن دکمه Shift به میانبرهای بالا می‌تونیم راه‌های ساده‌تری برای انتخاب یا Select متن داشته باشیم
    1. Shift + Command + Left Arrow برای انتخاب متن تا ابتدای خط (اگر کرسر در وسط یک خط قرار داشته باشد با این شرت‌کات از وسط تا
    ابتدای خط انتخاب می‌شود)
    2. Shift + Command + Right Arrow برای انتخاب متن تا انتهای خط (اگر کرسر در وسط یک خط قرار داشته باشد با این شرت‌کات از وسط
    تا انتهای خط انتخاب می‌شود)
    3. Shift + Option + Right Arrow برای انتخاب متن تا ابتدای کلمه‌ای که در حال نوشتن آن هستیم یا کرسر روی آن قرار دارد
    4. Shift + Option + Left Arrow برای انتخاب متن تا انتهای کلمه‌ای که کرسر روی آن قرار دارد
    5. Shift + Command + Up Arrow برای انتخاب تا ابتدای متن
    6. Shift + Command + Down Arrow برای انتخاب تا انتهای متن
    ین میانبرها در تمام نسخه‌های مک کار می‌کنه و همچنین اگر کیبورد بلوتوث به آی‌دیوایس (آیفون/آی‌پاد/آی‌پد) خودتون وصل کنید هم می‌تونید از این
    میانبرها در تایپ استفاده کنید
    همچنین اگر دقت کرده باشید دکمه‌های Command و Option و Shift و Control در مک نشانگرهایی مخصوص خودشون دارن که این نشانگرها به
    ترتیب زیر هستن
    ⌘ is the Command () key
    ⌃ is the Control key
    ⌥ is the Option (alt) key
    ⇧ is the Shift key
    ⇪ is the Caps Lock key
    fn is the Function key
